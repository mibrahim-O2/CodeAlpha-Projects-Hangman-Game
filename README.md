# Hangman Game
A simple **text-based Hangman game** in Python.  

Guess the word one letter at a time before you run out of attempts!

---

## Features

- Uses **Python basics**: strings, lists, loops, and conditionals  
- Randomly selects a word from a **predefined list**  
- **6 incorrect guesses** allowed  
- Interactive console input/output  
- Beginner-friendly and easy to understand  

---

1. Run the game:

```bash
python Hangman-Game.py
